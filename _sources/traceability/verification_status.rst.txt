Release Readiness
-----------------

For release, the following criteria must be met:

.. list-table:: Release Criteria
   :header-rows: 1
   :widths: 40 30 30

   * - Criterion
     - Target
     - Status
   * - Requirements approved
     - 100%
     - Check table above
   * - Requirements with tests
     - 100%
     - See Coverage page
   * - Tests implemented
     - 100%
     - Check table above
   * - Tests passed
     - 100%
     - CI/CD report
   * - Safety requirements verified
     - 100%
     - See ASIL tables above
   * - Orphaned requirements
     - 0
     - See Coverage page
